# DecideWise-AI

An AI-powered web app that helps users make better decisions by analyzing their situations and giving structured advice.

## Live Demo
https://decidewise-ai-so3a.onrender.com

## Features
- AI-based decision analysis  
- Pros & Cons breakdown  
- Final recommendation   
- Simple and modern UI  

## Tech Stack
HTML, CSS, JavaScript, Groq API

## How it works
User enters a situation → AI analyzes → returns structured decision output.

## Future Improvements
- Save history  
- Voice input  
- User accounts
